Git Cheatsheet

git config --global user.name blabla - встановити name користувача

git config --global user.email blabal@bla - встановити email користувача

git init - створити новий репозиторій

git status - дозводяє побачити. на якому етапі перебувають зміни

git add - додати змінені файли

git commit - створити новий коміт

git log - показати історію комітів

git checkout - повернутись до останнього коміту

git branch - показати всі гілки

git push - відправити локальні зміни до віддаленого репозиторію

git pull - отримати віддалені зміни у локальному репозиторії

git merge - зєднати дві гілки в одну

git clone - клонувати існуючий репозиторій

.gitignore - вказує на навмисно невідслідковані (untracked) файли, які git має ігнорувати

git revert - скасовує один коміт

git reset - повертається до попереднього стану проекту, видаляючи всі наступні коміти

git rebase - інтегрувати зміни від однієї гілки до іншої

git cherry-pick - вибрати коміт з однієї гілки та застосувати його до іншої

git stash -зберігає змінені відстежувані файли, виконує зміни та зберігає їх серед незавершених змін, які можна повторно застосувати в будь-який час.
